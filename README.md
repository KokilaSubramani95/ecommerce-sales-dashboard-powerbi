# E-Commerce Sales Dashboard (Power BI)

## Overview
This project presents an interactive **E-commerce Sales Dashboard** built using Power BI to analyze sales performance, customer behavior, and operational efficiency.

---

## Key Features
- KPI Tracking (Sales, Orders, AOV, Cancellation Rate)  
- Month-over-Month (MoM) Growth Analysis  
- Sales Trend Visualization  
- Category & State Performance Analysis  
- Order Status Funnel  
- Cancellation Analysis by Fulfilment  

---

## Key Insights
- Merchant fulfilled orders have a higher cancellation rate compared to Amazon  
- Top category contributes ~50% of total sales  
- Maharashtra generates the highest revenue among all states  
- Sales peaked in April and declined slightly afterward  

---

## Skills Demonstrated
- Data Modeling (Star Schema)  
- DAX Measures (MoM Growth, KPIs, Dynamic Calculations)  
- Data Visualization & UX Design
